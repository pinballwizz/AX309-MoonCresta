Moon Cresta for the AX309 Clone Dev Board.

Notes:
Setup for 1 player arcade controls (3 switches) (Left-Right-Fire).
Each switch is connected one side to GPIO Header J2 and other side to VCC 3.3v
Pin Header locations are specified in the "ax309.ucf" File.
Left and Right Audio outputs are on the J2 Header. Both Audio channels are required to get all sound effects.
See instructions in the "dac.vhd" file for connecting audio to ext amp/spkr.
Coin and Player1 Start controls are Keys 1 and 2 buttons on the AX309.

Build:
* Obtain correct roms file for Moon Cresta (mooncrstu.zip).
* Unzip rom files to mooncrstu folder inside the roms folder.
* Run the build roms script in the scripts folder.
* Open the MoonCresta_ax309 project file inside the build folder using Xilinx ISE 14.7 webpack or similar and compile.
* Generate Programming file with startup options set to use JTAG Clock.
* Program AX309 Board using Impact and the BIT file generated by XILINX Webpack.

Note: 
This game synthesis is affected by a sprite issue where all enemy sprites are identical which is not original game behaviour.
